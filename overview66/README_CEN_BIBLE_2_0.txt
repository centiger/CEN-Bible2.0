CEN Bible 2.0 통합 PWA

포함 모듈:
- 성경권별 한눈에보기
- 구약 39권 / 신약 27권 총 66권 데이터 반영
- 카드형 UI
- 접기/펼치기
- 원본 인포그래픽 보기 연결 구조
- 관련 자료 연결 프레임

GitHub Pages 업로드:
1. 압축을 풉니다.
2. index.html, manifest.json, service-worker.js, data 폴더, assets 폴더, icons 폴더를 저장소 루트에 업로드합니다.
3. 기존 파일은 같은 이름으로 덮어쓰기합니다.
4. 모바일에서는 기존 홈화면 아이콘을 삭제 후 다시 접속/추가하는 것이 안전합니다.
5. 반영이 늦으면 브라우저 캐시 또는 service-worker 캐시 문제일 수 있으니 새로고침/사이트 데이터 삭제 후 확인합니다.
