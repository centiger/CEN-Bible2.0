1 성경 읽기

      
    
        
  • 상단에서 성경권·장·번역본을 선택하여 본문을 읽습니다.
    • 
        
  • 본문 화면을 좌우로 밀면 이전 장 / 다음 장으로 이동합니다.
    • 
        
  • 화면을 한 번 터치하면 상단·하단 메뉴가 나타나고, 다시 터치하면 숨겨집니다.
    • 
        
  • 하단 4개 창을 이용해 서로 다른 성경 위치를 저장해 두고 빠르게 이동할 수 있습니다.
    • 
      

    


    
2 절 선택 기능

      
    
        
  • 성경 구절을 길게 누르면 절 선택 메뉴가 나타납니다.
    • 
        
  • 선택 모드에서는 다른 절을 터치하여 여러 절을 이어서 선택할 수 있습니다.
    • 
        
  • 선택한 절은 복사, 성구추가, 메모추가, 형광펜으로 활용할 수 있습니다.
    • 
        
  • 메뉴 밖을 터치하면 절 선택 메뉴가 닫힙니다.
    • 
      

    


    
3 나의 성구

      
    
        
  • 자주 보는 말씀을 카테고리별로 저장할 수 있습니다.
    • 
        
  • 카테고리를 직접 만들고, 이름을 수정하거나 위치를 이동할 수 있습니다.
    • 
        
  • 성경본문에서 절을 길게 누른 뒤 성구추가를 선택하면 저장할 카테고리를 직접 터치해 넣을 수 있습니다.
    • 
        
  • 성경 검색 결과에서도 원하는 구절을 나의 성구에 추가할 수 있습니다.
    • 
        
  • 저장한 성구는 다른 카테고리로 이동하거나 삭제할 수 있습니다.
    • 
      

    


    
4 메모

      
    
        
  • 구절별로 묵상, 설교 내용, 기도 제목 등을 기록할 수 있습니다.
    • 
        
  • 메모가 있는 절에는 표시가 나타납니다.
    • 
        
  • 메모를 누르면 해당 절 아래에 말풍선 형태로 내용이 펼쳐집니다.
    • 
        
  • 전체 메모 보기에서 저장된 메모를 한꺼번에 확인하고 검색할 수 있습니다.
    • 
      

    


    
5 성경읽기표

      
    
        
  • 읽은 장을 체크하여 권별·전체 읽기 진행률을 관리합니다.
    • 
        
  • 전체보기에서는 창세기부터 요한계시록까지 전체 진행률을 막대그래프로 확인합니다.
    • 
        
  • 권별보기에서는 구약/신약을 선택하고 특정 성경의 장별 읽기 상태를 확인합니다.
    • 
        
  • 읽은 장을 다시 터치하면 체크를 해제할 수 있습니다.
    • 
      

    


    
6 검색

      
    
        
  • 돋보기 버튼을 눌러 단어 또는 구절 일부를 검색합니다.
    • 
        
  • 검색 결과에서 본문 위치를 확인하거나 필요한 구절을 나의 성구에 저장할 수 있습니다.
    • 
        
  • 검색창의 초기화 버튼으로 검색어와 결과를 지울 수 있습니다.
    • 
        
  • 검색창에서 나가면 검색어와 결과가 자동으로 초기화됩니다.
    • 
      

    


    
7 예배용 세트

      

        
찬송가번호·제목 검색, 가사 보기

        
주기도문기도문 빠른 확인

교독문번호·제목 검색, 본문 보기

      

    


    
8 화면설정

      
    
        
  • 글자 크기, 줄간격, 절 간격을 조절할 수 있습니다.
    • 
        
  • 글자색과 배경색을 바꾸어 읽기 환경을 조정할 수 있습니다.
    • 
        
  • 나눔명조/나눔고딕 중 원하는 폰트를 선택할 수 있습니다.
    • 
      

    



    
9 데이터 내보내기·불러오기

      
    
        
  • 내보내기를 처음 누르면 먼저 휴대폰의 내 파일 앱에서 CEN Bible 폴더를 만들었는지 확인합니다.
    • 
        
  • 폴더를 만들지 않았다면 취소한 뒤 내 파일 > 내부저장공간 또는 원하는 저장 위치에 CEN Bible 폴더를 먼저 만드세요.
    • 
        
  • 폴더를 만든 뒤에는 앱의 안내에 따라 CEN Bible 폴더 권한을 허용하면 백업파일이 그 폴더에 저장됩니다.
    • 
        
  • 이후 내보내기는 저장된 권한을 확인한 뒤 같은 CEN Bible 폴더에 바로 백업합니다.
    • 
        
  • 파일명은 CEN_Bible_백업_날짜_시간.json 형식으로 저장되어 날짜별 구분이 쉽습니다.
    • 
        
  • 불러오기를 누르면 앱이 권한을 받은 CEN Bible 폴더의 백업파일 목록을 대화창으로 보여줍니다.
    • 
        
  • 목록에서 원하는 날짜의 백업파일을 선택하면 데이터가 복원됩니다. 백업파일이 없으면 CEN Bible 폴더에 백업파일이 없습니다.라고 안내합니다.
    • 
        
  • 휴대폰 보안정책상 폴더 권한이 해제되었거나 브라우저가 폴더 접근을 제한하면 다시 CEN Bible 폴더를 선택해야 합니다.
    • 
      

    


    
10 사용 권장

      
    
        
  • Chrome에서 홈 화면에 추가하면 앱처럼 빠르게 실행할 수 있습니다.
    • 
        
  • 권장 절차: 내파일에서 HTML 선택 → Chrome으로 파일 열기 → 메뉴 선택 → 홈 화면에 추가
    • 
        
  • 백업·복원 기능은 Chrome 또는 최신 Android 브라우저에서 사용하는 것이 가장 안정적입니다.
    • 
      

    


    
11 배포 유의사항

      
    
        
  • 무단 복제 및 배포를 금지합니다.
    • 
        
  • 본 앱은 개인의 영성 관리 및 비영리 학습 목적으로 제작되었으므로 상업적 이용을 금지합니다.
    • 
        
  • 이 앱에 사용된 성경 본문과 찬송가의 저작권은 각 출판사에 있으며, 저작권자의 요청 시 즉시 삭제될 수 있습니다.
